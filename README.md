# 校招实习群
校招实习群里会有对校招和实习有需求的校友，以及同一年毕业的期望进入北京互联网的同学们。大家可以彼此共享消息，互相帮助。
群名称中的“xxxx 届”表示 xxxx 年毕业的同学。
比如：2014 年入学的同学进入 “大工 2018 届北京互联网校招实习群”。
